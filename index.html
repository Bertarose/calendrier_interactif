À propos de ce document

Un horaire vivant, sans app, sans abonnement.

Ce guide présente le fonctionnement de l'horaire interactif du Festival Asiasie 2026 — une page web hébergée directement sur le site, installable comme une app sur téléphone, avec mise à jour en temps réel pendant l'événement.

01
Ce que l'horaire fait
⏱
Avant le festival
Compte à rebours en direct
Affiche jours / heures / minutes / secondes avant le 7 mai. Disparaît automatiquement à l'ouverture du festival.
📅
Navigation par journée
4 onglets — Jeu 7, Ven 8, Sam 9, Dim 10. Le jour en cours est marqué d'un ●. Un tap pour changer de journée.
🔴
Temps réel
Ligne de temps en direct
Une ligne rouge se déplace sur la liste selon l'heure exacte, se repositionnant à chaque minute automatiquement.
🟠
Temps réel
Bannière « En cours »
Quand un événement est en cours, une bannière orange apparaît en haut avec le nom, l'artiste et le lieu.
🎨
Filtres par lieu
Boutons pour filtrer par Scène, Atelier ou Centre Eaton. Chaque lieu a sa propre couleur. Idéal pour les bénévoles assignés à un poste fixe.
▶
Fiche détaillée par activité
Un tap ouvre un panneau qui glisse depuis le bas avec description complète, tags (gratuit, places limitées…) et lieu. Se ferme en glissant vers le bas.
♡
Favoris
Coups de cœur personnels
Chaque activité a un cœur. Un tap l'ajoute aux favoris. Un bouton regroupe tous les favoris par jour. Sauvegardé dans le téléphone.
📲
PWA
Installable comme une app
Android : bouton automatique. iPhone : guide en 3 étapes via Safari. Zéro App Store, zéro abonnement, zéro coût.
📶
PWA
Fonctionne hors ligne
Une fois installée, la page est mise en cache. Si le Wi-Fi du festival est lent ou absent, tout s'affiche quand même.
💚
Gratuit
Zéro coût, zéro abonnement
3 fichiers hébergés sur Webflow, Netlify ou GitHub Pages. Aucun service tiers, aucune base de données, aucune dépense récurrente.

05
Mises à jour du programme
✏️
Si le programme change
Kim modifie les données dans la section EVENTS du fichier HTML et re-téléverse le fichier sur Webflow. La mise à jour est visible immédiatement pour tout le monde, sans rechargement.

Règle importante : ne jamais partager une capture d'écran ou un PDF de l'horaire — ils peuvent être périmés. Toujours partager le lien asiasie.com/horaire.
