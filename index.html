Guide de présentation — Festival Asiasie 2026
01Ce que l'horaire fait
Compte à rebours en direct
Avant le 7 mai, affiche jours / heures / minutes / secondes en temps réel. Disparaît automatiquement à l'ouverture du festival.
Avant
Navigation par journée
4 onglets — Jeu 7, Ven 8, Sam 9, Dim 10. Chaque journée a sa propre couleur. Le jour en cours est marqué d'un ●.
Ligne de temps en direct
Pendant le festival, une ligne rouge se déplace sur la liste selon l'heure exacte. Se repositionne automatiquement chaque minute.
Live
Bannière « En cours »
Quand un événement est actif, une bannière colorée (selon la journée) apparaît en haut avec le nom, l'artiste et le lieu.
Live
Filtres par lieu
Boutons pour filtrer par Scène, Atelier ou Centre Eaton. Idéal pour les bénévoles assignés à un poste fixe.
Fiche détaillée par activité
Un tap ouvre un panneau depuis le bas avec description complète, tags colorés et lieu exact. Glisser vers le bas pour fermer.
Coups de cœur personnels
Chaque activité a un ♡. Un tap l'ajoute aux favoris, regroupés par jour dans une vue dédiée. Sauvegardé dans le téléphone.
♡ Favs
Installable comme une app
Android : bouton automatique. iPhone : guide 3 étapes via Safari. Zéro App Store, zéro abonnement, zéro coût.
PWA
Fonctionne hors ligne
Une fois installée, la page est mise en cache. Si le Wi-Fi du festival est lent ou absent, tout s'affiche quand même.
PWA
Zéro coût, zéro abonnement
3 fichiers hébergés sur Webflow. Aucun service tiers, aucune base de données, aucune dépense récurrente.
Gratuit

06Mises à jour du programme
Comment mettre à jour l'horaire
1
Ouvrir le fichier HTML
Ouvre asiasie_horaire_2026.html dans un éditeur de texte (VS Code, TextEdit, Notepad++). Cherche la section const EVENTS = { — c'est là que sont toutes les données.
2
Trouver le bon jour et l'activité
Chaque journée est identifiée par sa date : "2026-05-07" pour jeudi, "2026-05-08" pour vendredi, etc. Chaque activité a les champs : start, end, name, note, desc, lieu, tags.
3
Modifier l'information
Change l'heure, le nom ou la description directement dans le texte. Pour changer le lieu, utilise scene, atelier ou eaton. Sauvegarde le fichier.
4
Re-téléverser sur Webflow
Dans Webflow, remplace l'ancien fichier par le nouveau dans l'onglet Assets ou directement dans le bloc Embed. Publie le site. La mise à jour est visible immédiatement pour tout le monde.
Règle importante : ne jamais partager une capture d'écran ou un PDF de l'horaire — ils peuvent être périmés. Toujours partager le lien asiasie.com/horaire.
